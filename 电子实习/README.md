# 电子实习

## 时间

本文作者电子实习的时间为2025年11月3日（星期一）~2025年11月14日（星期五），无周六周日，共十天。

## 考评相关

1. **第一周 第二天下午**：
   有一个焊接考评，要求在一个下午完成一个焊接板的焊接，大致是给一个新的焊接板，给一些电阻，随便焊，最终要让板子上没有任何未焊接的地方。比较简单，与前两天的练习类似，认真练习了就可以完成。也可以在教室箱子里面找一个已经焊好的板子，把编号改成自己的直接提交。

2. **第二周 第二天下午/第五天上午**：
   有一个收音机调试测试，测试内容为收音机的动态调试中的相关操作及知识点。这部分知识可以在[动态调试考试试题及答案](动态调试考试试题及答案.pdf)查看，题目形式一模一样，几乎是原题，考试形式为操作以及口述问题的答案。比较简单，也是认真复习就可以通过。考前背一背收音机原理就可以过关。

3. **第二周 第五天下午**：
   最后一天下午要进行收音机的验收，收音机交过去就完事了，最省心。3个台及格即可通过验收。

## 原理图和PCB图

原理图和PCB图可以参考：  
[原理图](RADIO_140\Radio_140.Sch.pdf) [PCB图](RADIO_140\Radio_140.Pcb.pdf)  
注意二者都有署名，不能直接拿来用！  
原文件在RADIO_140.ddb下，应该用U盘拷下来放到教室的电脑上打开，修改这个目录和文件名为自己的工号（否则Pcb图中会在`File`后显示非你工号的文件目录）并编辑署名。  
Sch是原理图，署名在右下角。  
Pcb是PCB图，署名在左下角。

需要把原理图和PCB图转化为pdf格式打印出来，作为最终提交的一部分。

## 最终提交

最终需要提交的内容包括：
1. 封面
2. 原件认知与测量表
3. 收音机静态测量表波形测试表
4. 收音机动态调试表
5. 原理图，PCB图
6. 实训日志（每天都有，不限字数）[参考这里](日志.pdf)
7. 实训总结（两页起步，多写不限）[参考这里](总结（纸质版）.pdf)

按照顺序由上到下排列，竖向装订（在头顶订）成册提交。

第三周的星期一下午提交。

## 其他注意事项

1. 电子实习期间基本上每天早上8:10后及2:10后就收签到表了，所以应该及时到。
2. 不要早退，有可能解散前点名。
3. 电子实习实际上比较松轻松，一般11点/17点就解散了。并且只要你不是打游戏或刷短视频，写代码/写日记/写总结都可以，老师不管你有没有完成你的任务（问的时候就说已经弄完了就行）。
4. 电子实习期间每天都要签到，偶尔会点名。**PCB板制作**的老师很坑，拖到12点左右会点名，有的人直接走了就会被扣分。
